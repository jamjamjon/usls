# Installation

To use **usls** in your project, add it to your `Cargo.toml`.

=== "Crates.io"

    ```toml
    [dependencies]
    usls = { version = "latest-version", features = [ "cuda" ] }
    ```

=== "GitHub (Recommended)"

    ```toml
    [dependencies]
    usls = { git = "https://github.com/jamjamjon/usls", branch = "main" }
    ```
