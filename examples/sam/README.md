## Quick Start

```Shell

# SAM
cargo run -r --example sam

# MobileSAM
cargo run -r --example sam -- --kind mobile-sam

# EdgeSAM
cargo run -r --example sam -- --kind edge-sam

# SAM-HQ
cargo run -r --example sam -- --kind sam-hq
```
